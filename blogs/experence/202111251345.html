    # JavaScript

    # 数组备忘录

    • 添加、删除元素：

      • push(...items)——在尾部添加元素；
      • pop——从尾部提取一个元素
      • shift()——从头部提取一个元素
      • unshift(..items)——在头部添加元素
      • splice(start, delCount, ...items)——从start开始删除delCount个元素，并插入 items；
      • concat(..items)——复制当前数组的元素，并向其中添加items，如果 items 中任意一项是一个数组，就复制其元素；

    • 搜索元素

      • indexOf/lastIndexOf(item, from)——从索引start开始搜索 item，搜索到则返回该项的索引，否则返回-1
      • find/filter(func):通过func过滤元素，返回为true的第一个值/所有值；
      • includes(items, from)——如果数组中存在items元素则返回true,否则返回false;

    • 遍历元素

      • forEach(func)——对每个元素都调用一次func，不返回任何内容，中途不可退出循环，除非异常；

    • 转换数组

      • map(func)——根据对每个元素都调用一次func的结果来创建一个新数组；
      • sort(func)——对数组解析原位排序，直接修改数组；
      • reverse()——反转数组，并返回；
      • join(seperator)——根据分隔符seperator将数组转换成字符串并返回；
      • reduce/reduceRight(func, initial)——通过对数组每个元素调用func计算数组上的每个值，并在调用之间传递之间结果；

    • 截取数组

      • slice(start, end)——复制数组从索引start到索引end（不包括 end）的元素到一个新数组中，并返回该数组；

    • 判断

      • every(func)——对数组中每个元素都调用一次func，都满足func的规则时，返回true，否则返回false;
      • some(func)——对数组中每个元素都调用一次func，存在一个元素满足func的规则时，返回true，否则返回false;

    • 其他

      • Array.isArray(arr)——如果arr是一个数组实例，则返回true，否则返回false;

    有关完整列表，请参考手册 (opens new window)

    • 最笨的方法：用循环
    • [...new Set([].concat(arr1, arr2, ...))]
    • 什么时候用 for 循环，什么时候用 forEach 循环 Array.prototype.forEach() (opens new window): 为数组的每个元素执行一次callbak函数。
      • 除非抛出异常，不可中途退出循环（直接报错）；

      • 如果数组在循环过程中被修改了，则其他元素会被跳过

    # 计算机网络

    • 三次握手，四次挥手

    TCP 协议要提供可靠的数据传输服务，要事先通过三次握手来建立连接，再发送数据，最后通过四次挥手来释放连接这样一个过程。

    • 三次握手的意义
      要确保双方的发送功能和接收功能都正常

    • 四次挥手的意义
      第一次：客户端发送断开连接的请求，告诉服务器自己没有数据要发送了，这时候还可以接收服务器的数据。
      第二次：服务器收到断开请求后，知道客户端已经发完数据了，然后发送同意断开连接的请求，此时服务器还可以向客户端发送数据。
      第三次：服务器向客户端发送断开连接请求，告诉客户端我已经没有数据要发送了。
      第四次：客户端收到断开请求后，知道服务器已经报所有数据发完了，并发送同意断开连接的请求。
