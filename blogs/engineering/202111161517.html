    webpack的入门学习

    GuanDichen 2021/11/16 Webpack

    # 文档

    webpack中文网：https://www.webpackjs.com/ (opens new window)
    官方文档：https://webpack.js.org/ (opens new window)

    webpacks是一个打包模块化JavaScript的工具， 从 Webpack2 开始，已经内置了对 ES6、CommonJS、AMD 模块化语句的支持，但是webpack.cofig.js文件中只能用Node.js支持的CommonJS语法。

    webpack 4+以上版本，需要安装webpack-cli（用于在命令行中来运行webpack）

    # 配置选项

    webpack需要传入一个配置对象，官网的配置案例：
    webpack.config.js (opens new window)

    # 简易版的实战

    项目中用到的功能有：

    • ES6
    • PostCSS
    • SCSS
    • React框架

    webpack.config.js 配置如下：

    // 这里只能用 CommonJS规范导出一个描述如何构建`Object`对象
const path = require('path')
const HtmlWebpackPlugin = require('html-webpack-plugin');
const MiniCssExtractPlugin = require("mini-css-extract-plugin");


module.exports = {
  mode: 'development',
  // JavaScript 执行入口文件
  entry: './main.js',
  output: {
    // 把所有依赖的模块合并输出到一个 bundle.js 文件
    filename: 'bundle.js',
    // 输出文件都放到 dist 目录下
    path: path.resolve(__dirname, './dist')
  },
  // 源码跟踪，方便在浏览器里调试
  devtool: 'inline-source-map',
  // 开发服务器的配置，自动刷新
  devServer: {
    static: './dist',
  },
  module: {
    rules: [
      //   {
      //   // 用正则表达式去匹配要用该Loader去转换的CSS文件
      //   test: /\.css$/,
      //   use: [
      //     // 执行顺序是从后往前

      //     MiniCssExtractPlugin.loader,
      //     // 还可以通过 Object 的方式传入
      //     {
      //       loader: 'css-loader',
      //       // options: {
      //       // }
      //     }
      //   ]
      // },

      // babel语法转换
      {
        test: /\.js$/,
        use: {
          loader: 'babel-loader',
          // babel优先使用配置文件，如有的话
          options: {
            presets: ['@babel/preset-env', '@babel/preset-react'],
            plugins: []
          }
        }
      },

      // sccs文件转换css文件，执行顺序从后往前
      {
        test: /\.scss$/,
        use: [MiniCssExtractPlugin.loader, 'css-loader', 'postcss-loader', 'sass-loader'],
      }
    ]
  },
  plugins: [
    // 把CSS代码提取到独立的一个文件，style-loader是通过js把样式插入到dom里面
    new MiniCssExtractPlugin({
      // 生成的文件名
      filename: `[name]_[contenthash:8].css`,

    }),
    
    // 为我们快速创建一个HTML模板，并自动把webpack构建的bundle文件添加进来
    new HtmlWebpackPlugin({
      // 提供自定义HTML模板
      template: './template.html',
      // 文档标题
      title: 'Development',
    }),
  ]
}
