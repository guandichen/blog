    ES6的class语法

    GuanDichen 2022/2/18 JavaScript

    在 ES5 里面，我们可以通过构造函数和原型实现类的功能，ES6 里面提供了class的语法，实际上是这两者的语法糖，本质没有改变。

    # "class 语法"

    class Animal {
  constructor(name) {
    this.name = name;
  }
  run() {
    alert(`${this.name} is runniung!`);
  }
}

// 用法
let cat = new Animal("cat");
cat.run();

    class {...}实际上做了这些事情：

    • 创建一个名为Animal的构造函数，该函数成为类声明的结果，该函数的代码来自constructor方法，如果不写，它就会为空。3
    • 存储类的方法在prototype里面，例如Animal.prototype的run方法。

    类方法之间没有逗号，否则会报错

    在 JavaScript 里面，class是一种函数。

    class Animal {
  constructor(name) {
    this.name = name;
  }
  run() {
    alert(`${this.name} is runniung!`);
  }
}

alert(typeof Animal); // 'function'

// 更确切地说，等于 constructor方法
alert(Animal === Animal.prototype.constructor); // true

    # class 与普通函数的区别

    1. 通过 class 创建的函数具有内部特殊标记[[isClassConstructor]]: true，必须以new的方式来调用
    class User {
  constructor() {}
}

alert(typeof User); // function
User(); //  Uncaught TypeError: Class constructor User cannot be invoked without 'new'

    大多数 JavaScript 引擎中的类构造函数的字符串形式都以class ...开头

    alert(User); // class { ... }

    1. 类方法不可枚举，类定义将prototype中方法enumerable标志设置为false。
    2. 类总是使用use strict，类构造中所有的代码自动进入严格模式。

    # Getters/Setters

    跟对象字面量一样，类也有getter/setter，计算属性等

    class Animal {
  constructor(name) {
    this.name = name;
  }
  get name() {
    return this._name;
  }

  set name(value) {
    if (value.length < 4) alert("Name is too short!");
    else this._name = value;
  }
}

let animal = new Animal("Duck");

alert(animal.name); //  Duck

animal = new Animal(""); // Name is too short!

    # Class 字段

    我们可以在类里面定义属性。

    class Animal {
  name = "fish";
  swim() {
    alert(`${this.name} is swimming!`);
  }
}

new Animal().swim(); // fish is swimming!

    使用类字段绑定 this

    class SearchBar {
  constructor(value) {
    this.value = value;
  }

  onSubmit = () => {
    alert(this.value);
  };
}

let searchBar = new SearchBar("Mobile Phone");
setTimeout(searchBar.onSubmit, 1000); // Mobile Phone

    # 总结

    基本的类语法

    class MyClass {
  prop = value // 属性

  constructor(...) {}// 构造器

  method(...) {}  // method

  get something() {} // getter
  set something() {} // setter

  [Symbol.iterator]() {}  // 有计算名称的方法

  // ...
}
